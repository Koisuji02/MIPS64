# Description
Laboratories' exercises for the MIPS64 part of the course of "Architetture dei Sistemi di Elaborazione" (italian version of the course of "Computer Architectures") at Politecnico di Torino, Italy. The exercises are based on the MARS MIPS simulator, and they cover various aspects of MIPS64 assembly programming, including basic instructions, control flow, and system calls.
